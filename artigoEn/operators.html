Operators

In JavaScript, operators are symbols or keywords that allow you to perform various operations on variables and values. These operators can be classified into different types, depending on the operation they perform. The main categories of operators in JavaScript are: arithmetic, assignment, relational, logical, and ternary. Let's explore each of them with examples.

Arithmetic Operators

Arithmetic operators are used to perform mathematical operations with numbers. The main arithmetic operators are:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulo (remainder of division)
  • ++ Increment (increases the value by 1)
  • -- Decrement (decreases the value by 1)

Order of Arithmetic Operators

  1. Parentheses ( ): Parentheses have the highest precedence and are always evaluated first.
  2. Increment and Decrement (++, --): Increment (++) and decrement (--) operators have higher precedence, but their evaluation depends on whether they are pre-fixed (before the operand) or post-fixed (after the operand).
    • Pre-fixed (++a, --a): The increment or decrement occurs before any other operation.
    • Post-fixed (a++, a--): The increment or decrement occurs after the operation is executed.
  3. Multiplication, Division, and Modulo (*, /, %): These operators have the same precedence and are evaluated from left to right.
  4. Addition and Subtraction (+, -): Addition (+) and subtraction (-) have the lowest precedence but are also evaluated from left to right.

Assignment Operators

Assignment operators are used to assign values to variables. The basic assignment operator is =, but there are operators that combine assignment with mathematical operations. The main assignment operators are:

  • = Simple assignment
  • += Addition and assignment
  • -= Subtraction and assignment
  • *= Multiplication and assignment
  • /= Division and assignment
  • %= Modulo and assignment

Relational Operators

Relational operators are used to compare two values. They return a boolean value (true or false) based on the comparison. The main relational operators are:

  • == Equality (does not consider the type)
  • === Strict equality (considers the type)
  • != Inequality (does not consider the type)
  • !== Strict inequality (considers the type)
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

Logical Operators

Logical operators are used to perform logical operations on boolean values or to combine multiple boolean expressions. The main logical operators are:

  • && AND – Returns true if both expressions are true.
  • || OR – Returns true if at least one of the expressions is true.
  • ! NOT – Inverts the value of an expression (from true to false, or vice versa).

Ternary Operator

The ternary operator is a compact way to write an if-else statement. It evaluates an expression and returns a value based on a condition. Its syntax is:

condition ? value_if_true : value_if_false;

Where:

  • ? = IF
  • : = ELSE

Summary

In summary, here are the types of operators:

  • Aritmetic: Performs mathematical operations (+, -, *, /, %).
  • Assignment: Assigns values to variables (=, +=, -=, *=, /=, %=).
  • Relational: Compares two values and returns a boolean value (==, ===, !=, !==, >, <, >=, <=).
  • Logical: Performs logical operations on boolean values (&&, ||, !).
  • Ternary: A compact way to write if-else (condition ? value_if_true : value_if_false).

These operators are fundamental for creating logic and control in any JavaScript program.
