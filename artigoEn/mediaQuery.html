Using Media Query for Accessibility

The media query is a technique used in responsive design to apply different CSS styles depending on the characteristics of the user's device or screen, such as screen size, resolution, orientation, etc. It allows you to adjust the layout and content of a website so that it adapts properly to different devices (such as desktops, tablets, smartphones) and can also be used to improve the accessibility of a website.

In the context of accessibility, the media query can be used to adjust the content according to the specific needs of users with disabilities. This includes adjusting the layout for smaller screens, increasing font size, or changing contrast, among other things, without needing to alter the content structure.

How to Use Media Query to Improve Accessibility?

First, you declare the media query and the desired screen size (you can use max and min-width), then you repeat the properties previously used, but with the adjustments for the respective screens.

          
            @media (max-width: 600px) {
              body {
                font-size: 18px; /* Increases the font size to improve readability */
              }
            }

Detect User Preferences (Accessibility Preferences)

You can use the media query prefers-reduced-motion to adapt the site for users who prefer fewer animations or movements (an important feature for people with conditions like epilepsy or concentration difficulties).

          
            /* Detect if the user prefers less movement or animations */
            @media (prefers-reduced-motion: reduce) {
              .animated-element {
                animation: none; /* Disables animations */
              }
            }

Customize Layout According to Screen Orientation

You can adapt the layout based on the screen orientation (portrait or landscape). This can be helpful to improve the browsing experience depending on how the user is holding the device.

          
            /* Change the layout when the device is in landscape orientation */
            @media (orientation: landscape) {
              .container {
                grid-template-columns: 1fr 1fr 1fr; /* Displays three columns in landscape mode */
              }
            }
