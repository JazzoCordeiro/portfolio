Estructuras de Control de Flujo en JavaScript

Las estructuras de control de flujo son construcciones esenciales en cualquier lenguaje de programación que permiten controlar el orden y la ruta en la que se ejecutan las instrucciones del programa. Determinan cómo se ejecutará el código según condiciones específicas o cuántas veces se repetirá el código.

En otras palabras, las estructuras de control de flujo permiten cambiar dinámicamente la ejecución de un programa dependiendo de las condiciones (comandos de toma de decisiones) o repetir un bloque de código varias veces (comandos de bucle).

1. Estructuras de Decisión

Estas estructuras permiten que el programa elija entre diferentes caminos de ejecución según condiciones específicas.

- if y else

La declaración if ejecuta un bloque de código si la condición es verdadera. El else proporciona un bloque alternativo de código si la condición es falsa.


    let edad = 18;
    if (edad >= 18) {
      console.log("Eres un adulto.");
    } else {
      console.log("Eres menor de edad.");
    }

- else if

Si tienes más de una condición que verificar, puedes usar else if entre el if y el else. Esto permite verificar múltiples condiciones.


    let edad = 20;
    if (edad < 13) {
      console.log("Niño");
    } else if (edad >= 13 && edad < 18) {
      console.log("Adolescente");
    } else {
      console.log("Adulto");
    }

2. Estructuras de Bucle

Estas estructuras permiten repetir un bloque de código mientras una condición sea verdadera.

- for

El bucle for se utiliza cuando sabes cuántas veces deseas repetir un bloque de código. Es comúnmente utilizado cuando se itera sobre una lista o un array.


    for (let i = 0; i < 5; i++) {
      console.log(i); // Imprimirá de 0 a 4
    }

- while

El bucle while ejecuta un bloque de código mientras una condición sea verdadera. La condición se verifica antes de cada iteración, y el bucle puede detenerse si la condición se vuelve falsa.


    let i = 0;
    while (i < 5) {
      console.log(i);
      i++;
    }

- do...while

El bucle do...while funciona de manera similar al bucle while, pero la condición se verifica después de ejecutar el bloque de código. Por lo tanto, el bloque de código se ejecutará al menos una vez, incluso si la condición es falsa.


    let i = 0;
    do {
      console.log(i);
      i++;
    } while (i < 5);

3. Estructuras de Control de Flujo para Interrupción

Estas estructuras permiten alterar el comportamiento de los bucles o decisiones de maneras más avanzadas.

- break

La declaración break sale de un bucle, sin importar la condición del bucle. Es útil cuando deseas salir de un bucle antes de que la condición se vuelva falsa.


    for (let i = 0; i < 10; i++) {
      if (i === 5) {
        break; // Sale del bucle cuando i es igual a 5
      }
      console.log(i);
    }

- continue

La declaración continue hace que el bucle salte la iteración actual y pase a la siguiente sin ejecutar el código restante después del continue.


    for (let i = 0; i < 5; i++) {
      if (i === 3) {
        continue; // Salta la iteración cuando i es igual a 3
      }
      console.log(i);
    }

Resumen:

  • Estructuras de Decisión: Ejecutan diferentes códigos basados en condiciones (if, else, else if)
  • Estructuras de Bucle: Repiten un bloque de código mientras una condición sea verdadera (for, while, do...while)
  • Control de Flujo en los Bucles: Alteran el flujo de los bucles con break y continue

Estas estructuras de control de flujo son fundamentales para crear programas dinámicos que pueden reaccionar de manera diferente según las condiciones y repetir tareas cuando sea necesario. Hacen que tu código sea más eficiente y flexible, permitiéndote crear lógica compleja y rutas de ejecución adecuadas al problema que estás resolviendo.
