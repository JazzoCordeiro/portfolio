do e while em JavaScript

Os loops do e while em JavaScript são usados para executar repetidamente um bloco de código enquanto uma condição especificada for verdadeira. No entanto, há uma diferença fundamental entre eles: a verificação da condição.

Loop while

O loop while em JavaScript verifica a condição antes de cada iteração. Se a condição for verdadeira, o loop será executado. Se a condição for falsa desde o início, o loop nunca será executado.

Sintaxe do loop while:

while (condição) {
      // Código a ser executado enquanto a condição for verdadeira
    }

Exemplo de um loop while:

let contador = 0;
    
    while (contador < 5) {
      console.log(contador);  // Isso irá imprimir números de 0 a 4
      contador++;  // Incrementa o contador
    }

  • O loop será executado enquanto contador for menor que 5.
  • O contador++ dentro do loop incrementa o valor de contador em cada iteração.
  • Quando contador atingir 5, a condição contador < 5 se tornará falsa, e o loop será interrompido.

    • Loop do...while

    O loop do...while, ao contrário do loop while, garante que o bloco de código dentro do loop será executado pelo menos uma vez, independentemente da condição. Isso ocorre porque a condição é verificada após o bloco de código ser executado.

    Sintaxe do loop do...while:

    do {
      // Código a ser executado
    } while (condição);

    Exemplo de um loop do...while:

    let contador = 0;
    
    do {
      console.log(contador);  // Isso irá imprimir números de 0 a 4
      contador++;  // Incrementa o contador
    } while (contador < 5);

    Neste exemplo:

    • O bloco do será executado primeiro, imprimindo 0 na primeira iteração.
    • Depois, após a execução do bloco, verifica-se se a condição contador < 5 é verdadeira. Se for, o loop continua. Se a condição for falsa, o loop é interrompido.
    • Mesmo que a condição fosse inicialmente falsa, o loop ainda seria executado uma vez antes da verificação.

    Comparação: do...while vs while

    Característica Loop while Loop do...while
    Verificação da Condição Verifica a condição antes de executar o bloco de código Verifica a condição após executar o bloco de código
    Execução Garantida O código pode não ser executado se a condição for falsa inicialmente O código sempre será executado pelo menos uma vez
    Caso de Uso Útil quando você precisa verificar a condição antes de rodar o bloco de código Útil quando você precisa garantir que o código execute pelo menos uma vez, mesmo que a condição seja falsa

    Exemplo onde do...while é útil:

    Considere o caso em que queremos pedir para o usuário inserir um número até que ele insira um número válido maior que 0:

    let numero;
    
    do {
      numero = prompt("Digite um número maior que 0:");
    } while (numero <= 0);
    
    console.log("Você digitou um número válido:", numero);

    Neste exemplo, o código dentro do bloco do será executado pelo menos uma vez, pedindo ao usuário para digitar um número. Se o usuário digitar um número maior que 0, o loop será interrompido; caso contrário, ele pedirá novamente.

    Resumo:

    • Loop while: A condição é verificada antes de o código ser executado. Se a condição for falsa inicialmente, o código dentro do loop não será executado.
    • Loop do...while: O código é executado pelo menos uma vez antes de a condição ser verificada. Mesmo que a condição seja falsa inicialmente, o código dentro do loop será executado uma vez.

    Esses loops são úteis para cenários diferentes, dependendo de você querer que o loop execute pelo menos uma vez ou verificar a condição antes.
