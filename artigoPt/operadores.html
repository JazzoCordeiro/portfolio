Operadores

No JavaScript, os operadores são símbolos ou palavras-chave que permitem realizar várias operações em variáveis e valores. Esses operadores podem ser classificados em diferentes tipos, dependendo da operação que realizam. As principais categorias de operadores em JavaScript são: aritméticos, de atribuição, relacionais, lógicos e ternários. Vamos explorar cada um deles com exemplos.

Operadores Aritméticos

Os operadores aritméticos são usados para realizar operações matemáticas com números. Os principais operadores aritméticos são:

  • + Adição
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo (resto da divisão)
  • ++ Incremento (aumenta o valor em 1)
  • -- Decremento (diminui o valor em 1)

Ordem dos Operadores Aritméticos

  1. Parênteses ( ): Parênteses têm a maior precedência e são sempre avaliados primeiro.
  2. Incremento e Decremento (++, --): Os operadores de incremento (++) e decremento (--) têm maior precedência, mas sua avaliação depende de estarem pré-fixados (antes do operando) ou pós-fixados (depois do operando).
    • Pré-fixado (++a, --a): O incremento ou decremento ocorre antes de qualquer outra operação.
    • Pós-fixado (a++, a--): O incremento ou decremento ocorre depois que a operação é executada.
  3. Multiplicação, Divisão e Módulo (*, /, %): Esses operadores têm a mesma precedência e são avaliados da esquerda para a direita.
  4. Adição e Subtração (+, -): A adição (+) e a subtração (-) têm a menor precedência, mas também são avaliadas da esquerda para a direita.

Operadores de Atribuição

Os operadores de atribuição são usados para atribuir valores às variáveis. O operador de atribuição básico é =, mas há operadores que combinam atribuição com operações matemáticas. Os principais operadores de atribuição são:

  • = Atribuição simples
  • += Adição e atribuição
  • -= Subtração e atribuição
  • *= Multiplicação e atribuição
  • /= Divisão e atribuição
  • %= Módulo e atribuição

Operadores Relacionais

Os operadores relacionais são usados para comparar dois valores. Eles retornam um valor booleano (verdadeiro ou falso) com base na comparação. Os principais operadores relacionais são:

  • == Igualdade (não considera o tipo)
  • === Igualdade estrita (considera o tipo)
  • != Desigualdade (não considera o tipo)
  • !== Desigualdade estrita (considera o tipo)
  • > Maior que
  • < Menor que
  • >= Maior ou igual a
  • <= Menor ou igual a

Operadores Lógicos

Os operadores lógicos são usados para realizar operações lógicas em valores booleanos ou para combinar várias expressões booleanas. Os principais operadores lógicos são:

  • && E – Retorna verdadeiro se ambas as expressões forem verdadeiras.
  • || OU – Retorna verdadeiro se pelo menos uma das expressões for verdadeira.
  • ! NÃO – Inverte o valor de uma expressão (de verdadeiro para falso, ou vice-versa).

Operador Ternário

O operador ternário é uma forma compacta de escrever uma instrução if-else. Ele avalia uma expressão e retorna um valor com base em uma condição. Sua sintaxe é:

condição ? valor_se_verdadeiro : valor_se_falso;

Onde:

  • ? = SE
  • : = SENÃO

Resumo

Em resumo, aqui estão os tipos de operadores:

  • Aritméticos: Realizam operações matemáticas (+, -, *, /, %).
  • Atribuição: Atribuem valores às variáveis (=, +=, -=, *=, /=, %=).
  • Relacionais: Comparam dois valores e retornam um valor booleano (==, ===, !=, !==, >, <, >=, <=).
  • Lógicos: Realizam operações lógicas em valores booleanos (&&, ||, !).
  • Ternário: Uma forma compacta de escrever if-else (condição ? valor_se_verdadeiro : valor_se_falso).

Esses operadores são fundamentais para criar lógica e controle em qualquer programa JavaScript.
